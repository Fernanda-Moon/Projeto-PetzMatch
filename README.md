## 📖 Sobre o Projeto

PetzMatch é um protótipo funcional de um aplicativo mobile para conectar criadores e tutores de animais de estimação que desejam realizar cruzamentos responsáveis. O app simula toda a jornada do usuário: cadastro/login, exploração de pets com filtros, visualização detalhada de saúde e documentos, chat interno e gerenciamento do próprio perfil e pets.

Desenvolvido inteiramente com **HTML, CSS e JavaScript puro**, o PetzMatch utiliza dados mockados e `localStorage` para simular um backend, sendo ideal para estudos de front‑end, design responsivo e manipulação do DOM.

## ✨ Funcionalidades Principais

- **Tela de boas‑vindas** com modais de login e cadastro (validação de campos).
- **Home** com:
  - Localização simulada (Campina Grande, PB).
  - Carrossel de imagens (Bootstrap).
  - Categorias de pets (Felinos, Cães, Pássaros, Roedores, Peixes, Répteis).
  - Lista de "Pets próximos" baseada em localização.
 
- **Explorar** com filtros por tipo de animal e sexo.

- **Páginas de categoria** com cards dos pets e chips de filtro por sexo.
- **Detalhes do pet**:
  - Galeria de imagens (clique nas miniaturas para ampliar).
  - Cards informativos (Sexo, Idade, Raça, Peso).
  - Dados do tutor(a) e botões de contato.
  - Seção de saúde (Pedigree, Exames, Vacinas).
  
- **Chat** com interface inspirada em WhatsApp (conversas mockadas).

- **Perfil do usuário**:
  - Avatar, nome e telefone.
  - Lista dos próprios pets cadastrados.
  - Opções de menu (editar, privacidade, ajuda, sair).
  
- **Cadastro de pet**:
  - Formulário estilizado com seções para dados do dono e do pet.
  - Upload de fotos por arrastar ou seleção (preview em tempo real).
  - Validação de campos obrigatórios.
 
- **Navegação inferior** (bottom navigation) persistente em todas as telas principais.
